## rest and spread

rest

...rest get all items without having to pass them in individually

```javascript
// And the rest
const songs = ["Creep", "Everlong", "Bulls On Parade", "Song 2", "What I Got"];
const newSongs = [...songs];
console.log(newSongs);
// Logs all songs
```
